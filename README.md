# 浅仓透主题博客：透明的呼吸感 (Transparent Respiration) 🌊

> 它是“担当（Producer）”对偶像的观测记录，也是属于你个人心流随笔的栖息地。

这是一个基于 Astro 构建的高度定制化个人博客模板。它最初由强大的 Momo 极简框架分叉而来，在保留其快速、多合一的构建优势底座上，彻底重绘了视觉交互层，主打清冷、通透的**“呼吸感”美学**。

## 🎨 现行成果与视觉特性

### 1. 《星海夜雨》- 原生 3D 悬浮入口
抛弃了干瘪的页面加载过渡。当你首次潜入博客，迎接你的是深邃流动的宇宙星斑与细雨光柱。
- 你可以拖动鼠标感受毛玻璃卡片（Portal）产生阻力与 **3D 视差**。
- 一旦点击“去海边吗？”，视窗将打破物理边界极大扩充并褪去模糊，**“Dive-in 入海”**转场平滑进入主世界主页。 
- 高维度的视觉下拥有严丝合缝的性能管理（CSS 硬件级处理与无痕的生命周期注销）。

### 2. Glassmorphism 全局重塑
卡片、交互面甚至侧边栏，均剥离了生硬的轮廓。通过纯正的毛玻璃（Glassmorphism）特效加持，配合极简的无衬线字体（Noto Sans SC），整个信息结构就像沉在浅海中的玻璃碎片。不仅具备出色的纵深感，阅读体验也更显清冷。

### 3. 数据与精神流：Fragments (碎碎念) 
除了常规博客，额外开发了“碎碎念”流水线（`fragments`）。
同时配备了 `scripts/sync_obsidian.py`。你可以随时将本机的 Obsidian 日记、闪念无缝一键同步到博客内容库，形成如同海风一般的思绪流。

## 🔮 意象的延续 —— 未来构想 (Future Roadmap)

在当前“风、海、宇宙和毛玻璃”这一脉相承的视觉体系上，这里留下了一些能够让沉浸感达到极致的扩展路线图：

### 1. 「潮汐脉搏」- 环境氛围播放器 (Ambient Player)
不仅要有视觉的“自顾自漂浮”，还需注入听觉的环境感。
- **构想**：在侧边栏或角落的模糊面板内集成一个不突兀的 Web Audio 播放器。
- **形态**：没有生硬的播放进度条，而是跟随音乐律动呈现毛玻璃卡片的后置柔光或“水波纹泛动”。可以默认播放极细微的海浪白噪音（White Noise），营造深海观感。
  
### 2. 「透明的陪伴」- 沉浸式 Live2D 挂载展示
打破二次元与极简 UI 的割裂边界。
- **构想**：在博客特定的页面（如“关于我”或背景的最深处）放置一个 **Live2D 模型（如浅仓透）**。
- **形态**：配合当前架构，她不会突兀地盖在所有文字之上，而是被放置在全站的最底层**“深海”之中**。透过前方的毛玻璃博文卡片，你能隐约瞥见她在背景里安静地待着、偶尔被风吹起发梢。当我们不聚焦文字时，可以将卡片隐去，与偶像隔着屏幕完成静谧的对视。

### 3. 深海潜游模式 (Focus / Dive Mode)
- **构想**：当用户长时间（通过闲置监测算法）未滚动鼠标时，页面的非核心元素（导航栏、边距装饰等）慢慢融入背景隐去。让文字像真正悬浮在深蓝色宇宙中，将干扰降到 0 绝对值。

## 💻 快速安装部署

你可以十分轻松地在本地搭建这片海洋：
```bash
# 1. 抓取与环境预载
git clone <repository_url> transparent-respiration
cd transparent-respiration

# 2. 安装底层套件
pnpm install

# 3. 运行！
pnpm dev
# 服务器即刻起跑于 http://localhost:4321
```
