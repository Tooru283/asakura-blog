# 《星海夜雨》入口动画 - V2 (交互升级版) 🌊✨

看到初始版本的反馈，纯 CSS 在处理“逼真水滴”时确实容易变成死板的“水渍”，且剥离了交互后失去了“探索”的厚重感。这一次，我们将重点放在**“交互感 (Interactivity)”** 与 **“意象的具体化”** 上。将单向播放的动画，变为一次由用户主导的**“潜入”**仪式。

## User Review Required
> [!IMPORTANT]
> 1. **从“自动播放”升级为“点击潜入”**：这一次屏幕中央会有一块呼吸闪烁的、带有通明毛玻璃质感的“入口卡片”或纯粹的**文字按钮 (如 "潜入 / Dive In")**。只有当你点击它时，整个动画才会进入最终的高潮（玻璃碎裂/消融，背景放大，风浪卷走一切，进入主页）。
> 2. **水滴与风的呈现**：由于单纯的 CSS 阴影水滴不够生动，我将引入动态的星尘（CSS 粒子动画流）和真正的倾斜光绘雨丝。当你**鼠标悬停或移动时**，雨丝的倾斜角度和主毛玻璃卡片会产生轻微的 3D 视差 / 倾斜摇摆（随鼠标而动），这就赋予了“风”的灵魂。
> 3. 是否同意将全屏的毛玻璃改为**中央聚焦的巨大毛玻璃视窗**（视觉主体更为明确）？

## Proposed Changes

### Components Layer

#### [MODIFY] src/components/SplashScreen.astro
彻底重构代码，增加交互与多层次的深度视觉：
- **宇宙层 (Universe)**：基于 `box-shadow` 生成多层流动的微光粒子，增加粒子的发光度（box-shadow blur radius）。
- **雨丝层 (Rain)**：使用 SVG 或高对比度的 CSS 细线生成不断下坠的“光雨”。
- **交互毛玻璃视窗 (Glass Portal)**：
  - 在屏幕正中央放置一个巨大的圆形或圆角矩形的极度模糊视窗 `backdrop-filter: blur(20px)`。
  - 中心是一个**呼吸态的按钮**（如 `Dive in`，搭配浅海蓝光晕）。
- **鼠标视差 (Parallax & Wind)**：
  - 增加 JS 监听 `mousemove`：当鼠标滑动时，中心的毛玻璃视窗发生轻微 3D 偏转（透视效果），雨丝的方向也会随着鼠标稍微发生角度偏移（模拟你的鼠标就是“风”）。
- **点击潜入特效 (Click to Enter)**：
  - 监听点击事件：一旦点击按钮，中心的玻璃视窗急剧**放大 (scale: 5) 并变透明**，就好像我们穿透了这面沾满雨水的玻璃。
  - 随后背景消融，平滑衔接到博客的正文。

---

### Layout Integration

#### [MODIFY] src/layouts/Layout.astro
- `<slot />` 的进场方式不变。由于这次是点击触发，因此在点击“潜入”后，通过触发在 `Layout` 和 `SplashScreen` 间的 class 切换，使得博客正文内容由小变大、由暗变亮（从深海浮出水面）。
